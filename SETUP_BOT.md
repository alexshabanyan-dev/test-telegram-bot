# Настройка бота для отправки уведомлений

## Шаг 1: Получение токена бота

1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте команду `/newbot`
3. Следуйте инструкциям для создания бота
4. Скопируйте токен, который выдаст BotFather

## Шаг 2: Добавление токена в .env

На сервере откройте файл `.env`:

```bash
nano ~/test-telegram-bot/.env
```

Добавьте строку с токеном:

```
BOT_TOKEN=8381346317:AAE0W8L5iBk3F3tpVc6ewgbg-Tqmq_crtTE
```

**Важно:** Замените токен на ваш собственный, если используете другой бот.

## Шаг 3: Добавление бота в группу/канал

1. Откройте группу/канал, куда нужно отправлять уведомления
2. Нажмите "Добавить участников"
3. Найдите вашего бота по username (который вы указали при создании)
4. Добавьте бота в группу

## Шаг 4: Перезапуск бота

```bash
cd ~/test-telegram-bot
git pull
sudo systemctl restart telegram-monitor.service
```

## Проверка работы

Проверьте логи:

```bash
sudo journalctl -u telegram-monitor.service -f
```

Должно появиться:
- `✅ Бот подключен для отправки уведомлений`
- При отправке уведомления: `✅ Уведомление отправлено в Telegram от бот`

## Безопасность

⚠️ **Важно:** Никогда не публикуйте токен бота в открытом доступе!
- Токен дает полный контроль над ботом
- Файл `.env` уже добавлен в `.gitignore` и не попадет в репозиторий
- Не коммитьте `.env` файл в git
